# enterprise-system-services-deployment
Xây dựng, triển khai và bảo mật hạ tầng mạng nội bộ tập trung cho doanh nghiệp nhỏ (10 nhân sự) trên môi trường ảo hóa VMware. Dự án bao gồm quản trị Windows Server (AD DS, DNS, DHCP Relay, GPO), triển khai dịch vụ Web/Mail nội bộ và phân hoạch VLAN phòng ban.

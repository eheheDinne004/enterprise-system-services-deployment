## Tổng quan

- **Mục tiêu:** Thiết kế và triển khai hệ thống mạng nội bộ chuẩn hóa, quản trị tập trung và bảo mật cho doanh nghiệp nhỏ (10 nhân sự), giải quyết tình trạng mạng "chắp vá", xung đột IP và rủi ro rò rỉ dữ liệu.
- **Đối tượng & Phạm vi:**
  - Mô phỏng trên **VMware** cho 5 phòng ban (Ban Giám đốc, HR, IT, Kinh doanh, Kế toán).
  - Triển khai phân đoạn mạng (**VLAN/Subnet**) và các dịch vụ cốt lõi: **Active Directory (AD DS), DNS, DHCP Relay, Web Server (IIS), Mail Server (MDaemon), GPO & Backup**.
  - *(Không bao gồm hạ tầng Internet thực tế, VPN đa chi nhánh hay SIEM/SOC quy mô lớn).*
- **Ý nghĩa:** Tối ưu hóa chi phí vận hành, chuẩn hóa kiến trúc bảo mật từ gốc (phân quyền tài nguyên, kiểm soát truy cập), đảm bảo tính sẵn sàng dữ liệu và là nền tảng thực hành quản trị hạ tầng CNTT thực tế.
